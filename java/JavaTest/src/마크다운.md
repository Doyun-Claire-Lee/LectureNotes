# 마크 다운
마크 다운은 마크업 언어의 일종으로 간단한 문법을 사용해 마크업 결과를 만드는데 사용합니다.

두번째 문단입니다. (문단 만들 때 - 엔터 두번)  
앞 문장에 스페이스 두번 - 줄바꿈


## 마크 다운 요소
- 제목
- 항목
- 이미지
    - 강아지 (Tap 대용 스페이스 4번)
    - 고양이
- 링크
- 코드
- 기타 등등
