HTML
- Hyper Text Markup Language
	- Hyper Text: 링크
	- Markup Language: 표기 형태
- SGML에서 표현(화면에 보여주기)에 관련된 기능들만 발췌하여 모아놓은 언어, 사용법이 쉬움


Markup Language
- 마크업 표기법(<>)을 사용해서 정의된 언어
- SGML > 메타 언어(다른 언어를 만들어 낼 수 있는 기능을 가진 언어) > HTML, XML


HTML 1.0 > HTML 2.0 > ... > HTML 4.01(***) > XHTML 1.0 > HTML 5


HTML 개발 환경
- 텍스트 편집기
- 메모장
- SublineTest, Atom, Bracket, Visual Studio Code...
- IDE > Visual Studio
- 브라우저(실행, 디버깅)


HTML 실행 환경
- 브라우저(실행)


HTML 실행 방법
1. 로컬 실행
	- 파일을 직접 브라우저를 통해서 연결 + 실행
	- 쓰지말기, 서버(?)를 배울 때 로컬 실행 방법으로는 동작이 안되기 때문..
2. 서버 실행(*****)
	- 파일을 서버를 통해서 브라우저에게 전달 + 실행
